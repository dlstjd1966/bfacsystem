추후 개발 계획

    결제요청
    각 사무실에는 메신저처럼 보임 메신저에 당일 결제금액을 전송
    마스터계정에는 메신저로 온 게시글이 한 화면에 보임 (카카오톡 단체채팅처)

    엑셀 업로드 거래내역 종합
    오너클랜 대량주문서 엑셀을 업로드하면
    자동으로 주문내역이 기록됨 (필요한 기능일까?)

    단가표
    말그대로 단가표임
    상품 작업 후 오너클랜 기본양식에 맞게 엑셀을 올리면
    자동으로 단가표 생성 기능

    시스템 건의사항
    기본적으로 시스템으로 움직여야 하기 때문에
    각 사무실간 소통은 불허.
    건의함만 제작하여 효용성 있는 프로그램만 수용 및 제작계획 수립을 위함

    주문통합관리
    추후 오너클랜과 동시에 다른 마켓에 상품을 올리게 된다면
    혹은 오너클랜의 여러 계정을 동시에 운영하게 된다면
    사용하게 될 기능이다.
    여러 마켓 혹은 여러 계정의 주문서를
    자동으로 긁어 모아서 엑셀시트로 출력해주는 기능

    번외기능

  1. 자동 품절 관리
    2. 각 거래처의 검색엔진을 이용하여
    단가표의 상품명을 자동으로 검색하여
    품절,단종,없는상품 등을
    오너클랜 상품코드와 함께 출력해주는 기능
